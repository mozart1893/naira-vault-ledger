# 🛡️ Admin Portal - Quick Start

## ⚡ Start in 3 Steps

### **Step 1: Install Dependencies** (First time only)
```bash
cd admin-portal
npm install
```

### **Step 2: Start Admin Portal**
```bash
cd admin-portal
npm run dev
```

### **Step 3: Access**
```bash
open http://localhost:3002
```
